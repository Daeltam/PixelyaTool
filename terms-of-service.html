Rules

This bot does not require a lot of rules, but here they are :

  • Do not ruin this for others, for example by rendering the bot unusable.
  • Do not spam the commands, especially the download commands. They require time to process, and the server can have a hard time handling the files.
  • If a command didn't work, wait for several minutes before trying again, and make sure the error does not come from your part.
  • The bot can sometimes crash if the file was too heavy for the server or Discord to handle. In this case it should automatically restart. Do not try using the same command again, it will result in the same issue.
